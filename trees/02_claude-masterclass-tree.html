Module Summary
  • Five mistakes, one root cause: search-engine mindset. Brief-style prompts, visual structure, specific roles, iterative workflow, and full context loading — each fix is measurable; applying all five is compound.
  • Eight techniques build on each other: four-block formula → XML → prefill → constraint stacking → chaining → style example → motivated instructions → iterative refinement. The three-layer stack (XML + prefill + constraints) covers 90% of use cases.
  • Routines: start local, graduate to remote. The env trap (secrets in .env = silent failure) and statelessness (every run starts from zero) are the two architecture facts that cost the most hours when missed.
  • Claude Design: design system first, always. Tweaks → Markup → Edit before the main prompt. Full page regeneration costs 10× a section — never iterate on the whole page.
  • Model selection: Sonnet (default, $3/$15) → Opus (final review + effort parameter, $5/$25) → Haiku (sub-agent scale, $1/$5). Extended Thinking reduces caching efficiency — toggle off for repetitive identical workflows.
  • Claude vs. Gemini: complementary. Gemini for video, multimodal, and cost-efficient volume; Claude for code reliability, agent coherence, transparency, and correctness-critical work. The optimal production stack uses both.
