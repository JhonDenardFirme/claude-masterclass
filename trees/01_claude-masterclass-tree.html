Module Summary
  • Model cascade — Sonnet by default, Opus when depth falls short, Haiku for bulk. In Claude Code: Opus plans, Sonnet builds.
  • The highest-leverage features by effort: Extended Thinking (on for ~20% of queries), Memory (3-conversation foundation + hygiene), 1M token context window (76% edge accuracy).
  • Projects + Artifacts shift the paradigm from text generation to persistent workspace and real file delivery.
  • Prompt craft stack: five-part framework → XML tags → meta-prompting → interview-first. Each layer adds precision.
  • Claude Code core: /init → claude.md (negative constraints first) → Opus plan → Sonnet build → /plan before complex tasks → /compact mid-session → fresh session after three failed corrections.
  • Cowork is infrastructure: context files + per-workflow Projects + scheduled tasks turn one-time work into permanent, self-running automation.
